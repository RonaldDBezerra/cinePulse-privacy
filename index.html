🎬 CinePulse

Política de Privacidade e Exclusão de Conta

📌 Sobre o aplicativo

O CinePulse é um aplicativo mobile que permite aos usuários descobrir filmes e séries em alta, utilizando dados fornecidos pela API do The Movie Database (TMDB).

🔐 Dados coletados

O aplicativo pode coletar os seguintes dados:

  • Email e senha (para autenticação)

Esses dados são armazenados e gerenciados de forma segura através do Firebase Authentication.

⚙️ Uso dos dados

Os dados coletados são utilizados exclusivamente para:

  • Autenticação do usuário
  • Funcionamento do aplicativo

O CinePulse não compartilha dados pessoais com terceiros.

🗑 Exclusão de conta

Você pode solicitar a exclusão da sua conta a qualquer momento.

Como solicitar:

  1. Envie um e-mail para: devronalddbezerra@gmail.com
  2. Use o assunto: "Excluir conta"
  3. Informe o e-mail utilizado no cadastro

A exclusão será realizada em até 7 dias.

O que será excluído:

  • Conta do usuário
  • Dados de autenticação vinculados

Retenção de dados:

Não mantemos dados pessoais após a exclusão da conta, exceto quando necessário por obrigações legais.

🔒 Segurança

Os dados dos usuários são protegidos utilizando práticas modernas de segurança e infraestrutura do Firebase.
